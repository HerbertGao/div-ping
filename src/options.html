Webhook配置

支持的变量

在Webhook URL、请求头和请求体中可以使用以下变量:

{{projectId}}      - 项目ID
{{projectName}}    - 项目名称
{{url}}            - 页面URL
{{selector}}       - CSS选择器
{{oldContent}}     - 旧内容
{{newContent}}     - 新内容
{{timestamp}}      - ISO时间戳

示例配置

GET请求示例:

URL: https://api.example.com/notify?project={{projectName}}&old={{oldContent}}&new={{newContent}}

POST请求示例:

URL: https://hooks.slack.com/services/YOUR/WEBHOOK/URL
Body: {
  "text": "{{projectName}} 检测到变化: {{newContent}}"
}

自定义请求头示例:

Headers: {
  "Authorization": "Bearer YOUR_TOKEN",
  "X-Project": "{{projectName}}"
}
